1단원

문장 조각을 순서대로 세우기

1-1 Block Ordering

1단계. 영어 문장은 조각을 아무 순서로나 놓지 않습니다.

2단계. 먼저 중심 뼈대를 세우고, 설명 조각을 알맞은 자리에 붙입니다.

3단계. 이제 block들을 자연스러운 영어 순서로 배열해봅니다.

1-2 Block Ordering, No Korean

1단계. 이번에는 한국어 뜻 없이 영어 조각만 보고 순서를 잡습니다.

2단계. 단어 뜻보다, 주어·동사·목적어·설명 조각의 자리를 봅니다.

3단계. 이제 영어 구조만 보고 자연스러운 문장을 만들어봅니다.

2단원

문장 안의 중심축 보기

2-1 Fragment Linking & Pivot

1단계. 문장 안에는 여러 조각을 연결하는 중심 단어가 있습니다.

2단계. 이 중심 단어를 pivot처럼 보고, 앞뒤 조각이 어떻게 붙는지 봅니다.

3단계. 이제 어떤 단어를 중심으로 의미가 연결되는지 찾아봅니다.

2-2 Block Conversion

1단계. 따로 떨어진 조각들도 서로 겹쳐 하나의 문장이 될 수 있습니다.

2단계. 중요한 것은 조각을 그냥 붙이는 게 아니라, 자연스럽게 포개는 것입니다.

3단계. 이제 겹치는 조각들을 하나의 문장으로 조립해봅니다.

2-3 Pivot Split

1단계. 한 단어는 문장 뼈대에도 속하고, 수식 조각에도 속할 수 있습니다.

2단계. pivot을 기준으로 문장을 나누면 겹친 구조가 보입니다.

3단계. 이제 중심 단어를 기준으로 문장 조각들을 분리해봅니다.

3단원

같이 읽어야 하는 덩어리 보기

3-1 Chunk-based Translation

1단계. 영어는 단어 하나씩이 아니라, 같이 붙어 다니는 덩어리로 읽어야 합니다.

2단계. afraid of, depend on, take care of 같은 표현은 한 chunk입니다.

3단계. 이제 주어진 chunk를 중심으로 문장을 해석해봅니다.

3-2 Chunk Identification

1단계. 이번에는 chunk가 미리 주어지지 않습니다.

2단계. 문장 안에서 같이 읽어야 하는 단어 묶음을 직접 찾아야 합니다.

3단계. 이제 문장 속에서 의미가 붙어 있는 chunk를 골라봅니다.

3-3 Pivot + Chunk Translation

1단계. 하나의 중심 단어에 여러 chunk가 붙을 수 있습니다.

2단계. built by, built for처럼 같은 동사에 다른 방향의 의미가 붙습니다.

3단계. 이제 pivot에 붙은 chunk들을 나누어 해석해봅니다.

3-4 Multi-Chunk Long Sentence Translation

1단계. 긴 문장에는 chunk가 하나만 있는 것이 아닙니다.

2단계. 여러 chunk를 각각 잡아야 전체 문장이 덜 복잡해집니다.

3단계. 이제 긴 문장을 chunk 단위로 나누어 해석해봅니다.

4단원

긴 문장을 fragment로 보기

4-1 Guided Fragments

1단계. 긴 문장은 하나의 줄이 아니라 여러 의미 조각으로 이루어져 있습니다.

2단계. fragment는 문장 안에서 의미를 가진 작은 조각입니다.

3단계. 이제 제시된 fragment들을 보며 전체 문장의 뜻을 조립해봅니다.

4-2 Fragments to Sentence Reconstruction

1단계. 이번에는 fragment들을 보고 원래 문장을 다시 만듭니다.

2단계. 각 조각이 주어인지, 동사구인지, 수식어인지 봐야 합니다.

3단계. 이제 fragment들을 자연스러운 영어 문장으로 재조립해봅니다.

4-3 Fragments to Sentence Reconstruction 심화

1단계. fragment들은 단순히 나열된 조각이 아니라 서로 안기고 겹칩니다.

2단계. 어떤 조각은 명사를 꾸미고, 어떤 조각은 문장 전체를 설명합니다.

3단계. 이제 fragment들의 포함 관계를 보며 문장을 완성해봅니다.

5단원

문장 구조 패턴 보기

5-1 Parallel Listing

1단계. A, B, and C처럼 나열된 말들은 보통 같은 층위에 있습니다.

2단계. 중요한 것은 각각의 뜻보다, 이들이 어떤 공통 주제로 묶이는지 보는 것입니다.

3단계. 이제 나란히 놓인 말들과 그 공통 범주를 찾아봅니다.

5-2 Parallel Antithesis Type A

1단계. 대조 문장은 두 대상과 두 속성이 서로 마주 봅니다.

2단계. A는 X이고, B는 Y인 구조를 잡으면 문장이 선명해집니다.

3단계. 이제 무엇과 무엇이 대조되는지, 각각 어떤 특징을 갖는지 찾아봅니다.

5-22 Parallel Antithesis Type B

1단계. 대조를 읽을 때는 먼저 비교되는 두 대상을 찾아야 합니다.

2단계. 속성보다 먼저 A와 B가 무엇인지 잡는 것이 중요합니다.

3단계. 이제 문장에서 대조되는 두 축을 찾아봅니다.

5-23 Parallel Antithesis Type C

1단계. 대조 문장은 앞부분의 구조를 뒤에서도 이어갑니다.

2단계. 의미도 반대되어야 하고, 문장 모양도 비슷하게 맞아야 합니다.

3단계. 이제 앞 문장의 구조를 보고 뒤에 올 표현을 골라봅니다.

5-3 Restatement Within Sentence

1단계. 한 문장 안에서 같은 대상을 다른 말로 다시 설명할 수 있습니다.

2단계. 어려운 말 뒤에 쉬운 설명이 콤마로 붙는 경우가 많습니다.

3단계. 이제 한 문장 안에서 같은 개념을 가리키는 표현들을 찾아봅니다.

5-32 Restatement Between Sentences

1단계. 글에서는 같은 개념을 똑같은 말로 반복하지 않습니다.

2단계. 다음 문장에서 다른 표현으로 다시 부를 수 있습니다.

3단계. 이제 문장 사이에서 같은 개념이 어떻게 바뀌어 표현되는지 찾아봅니다.

5-32 Frame Echo Completion

1단계. 어떤 문장은 의미뿐 아니라 구조의 틀도 반복됩니다.

2단계. A와 B의 대조, 긍정과 부정의 방향이 그대로 이어져야 합니다.

3단계. 이제 앞 문장의 구조와 의미를 함께 유지하는 문장을 골라봅니다.

5-33 Restatement Cross-sentence A/B

1단계. 앞 문장에 두 개념이 나오면, 다음 문장에서 둘 다 다른 말로 이어질 수 있습니다.

2단계. A가 무엇으로 바뀌고, B가 무엇으로 바뀌는지 추적해야 합니다.

3단계. 이제 문장 사이에서 A/B 두 축이 어떻게 다시 표현되는지 찾아봅니다.

5-4 Emphasis & Inversion

1단계. 영어는 강조하려고 문장 순서를 바꾸거나 특수한 틀을 씁니다.

2단계. It was ~ that, only after, never 같은 표현은 강조된 부분을 보여줍니다.

3단계. 이제 강조·도치 문장을 일반적인 의미 순서로 풀어봅니다.

6단원

구조와 조각을 함께 보기

6-1 Pleks Final

1단계. 긴 문장에는 fragment, chunk, pivot, 병렬, 대조, 재진술이 함께 들어 있습니다.

2단계. 먼저 큰 구조 패턴을 보고, 그다음 작은 의미 조각으로 나눕니다.

3단계. 이제 문장을 자신이 읽기 편한 단위로 나누고, 구조를 함께 표시해봅니다.
